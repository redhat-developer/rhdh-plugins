Identify / Group Calls

Call Triggered When ID Used Traits Notes
identify First time user data with userID is available in session userID
company
 Runs once per session per user
group First time user data with accountID is available accountID (as groupId)
ebs: account number string (when present)
 Runs once per session per account

Common Properties

Every track call includes the following properties:

  • category: Developer Sandbox|<Section>
  • regions: sandbox-<section-lowercase>
  • text: the item name
  • href: destination URL (may be empty for some CTAs)
  • linkType: cta or default
  • internalCampaign: present for some Catalog CTAs (e.g., Ansible, OpenShift)
