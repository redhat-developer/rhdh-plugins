# Boost Workspace

This workspace contains the Boost plugin family for Red Hat Developer Hub.

Boost is a rewrite of the [Augment](../augment/) plugin.

## Development

```bash
# Install dependencies
yarn install

# Start the dev server
yarn start

# Run tests
yarn test:all

# Build all plugins
yarn build:all
```
